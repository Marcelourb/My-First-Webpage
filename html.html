What is HTML?



Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for web pages and applications. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

HTML plays a couple of significant roles in a web page. While HTML provides the bones of a web page, CSS provides the skin, and JavaScript provides the brains. First, we use the structure created by our HTML code to reference, enhance, and manipulate elements on a web page using CSS and JavaScript. For instance, you could use HTML to mark all of the headings on a web page, and then use CSS to specify the font, size, and color you want to apply to those headings to reflect your organization’s branding, or simply a visual design developed for the site.


Here is how the basic code looks like on HTML and how it will be displayed in the website:


<!DOCTYPE html>
<html>
<body>
<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
