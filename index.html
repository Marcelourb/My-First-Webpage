Client Side Programming



Hello there!

Have you ever heard about Client-side Programming and know what does it means?

Client-side Programming is the program that runs on the client machine (browser) and deals with the user interface/display and any other processing that can happen on the client machine such as:

  • Reading/writing cookies
  • Interact with temporary storage
  • Make interactive web pages
  • Interact with local storage
  • Sending request for data to server
  • Work as an interface between server and user
Here in this website, I will present some of the Programming languages used on Client-side Programming, explaining what they are used for and showing some features.
